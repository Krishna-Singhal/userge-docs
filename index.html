    Introduction

    Welcome to the Userge API. You can use our API to access Userge API endpoints, which can get information on Bans and tokens in our data.

    We have language bindings in Python. You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    Installation

    You can install our pypi libaray via

    pip install -U UsergeAntiSpamApi

    Authorisation

    You need to create a API Token for using library or for doing request on API. For more info on how to create api key, click here.

    Permissions

    Each endpoint (except /version and /stats) requires a certain permission level. The required level is mentioned in the documentation of the endpoint.

    The API has 2 Type of Level:

    Type Description
    User Type User can have only can_query rights.
    Admin Type Admin can have all rights.


    The API has 1 type of User Right and 4 type of Admin Rights.

    Rights Description
    can_query Can query Bans.
    can_ban Can add bans in Userge Api.
    can_unban Can delete bans either Ban is given by him or not.
    can_promote Can promote any User token to Admin type.
    can_demote Can demote any Admin token to User type.


    Getting your own Token

    User

    A token with Type User have only can_query permission level.

    /ban

    Request Path
    GET /ban

    /token

    Request Path
    DELETE /token

    /

    Request Path
    GET /version
    GET /stats

    Admin

    A token with Type Admin can do following according to rights.

    Rights Request Path
    can_ban POST /ban and /updateban
    can_unban DELETE /ban
    can_promote POST /promotetoken
    can_demote POST /demotetoken

    /ban

    Request Path
    GET /ban

    /token

    Request Path
    POST /token

    /

    Request Path
    GET /version
    GET /stats

    Bans

    Getting a specfic ban

    Check the ban status of a specific User.

    HTTP Request

    GET https://api.userge.live/ban?api_key=$token&user_id=$id

    URL Parameters

    Parameter Description
    api_key Your Api key
    user_id The ID of the User to check status

    Getting all bans

    This will return List of all Bans.

    HTTP Request

    GET https://api.userge.live/ban?api_key=$token

    URL Parameters

    Parameter Description
    api_key Your Api key

    Required Permission Level

    Admin

    Read more about the permission levels here.

    Adding a ban

    HTTP Request

    POST https://api.userge.live/ban

    Headers

    Parameter Description
    Content-Type application/json

    URL Parameters

    Parameter Description
    api_key Your Api Key
    user_id The Id of the User
    reason Reason for ban

    Updating a ban

    This method will Update a Old Ban.

    HTTP Request

    POST https://api.userge.live/updateban

    Headers

    Parameter Description
    Content-Type application/json

    URL Parameters

    Parameter Description
    api_key Your Api Key
    user_id The Id of the User
    reason Reason for ban

    Required permission level

    Admin with can_ban Rights.

    Deleting a Ban

    HTTP Request

    DELETE https://api.userge.live/ban

    Headers

    Parameter Description
    Content-Type application/json

    URL Parameters

    Parameter Description
    api_key Your Api Key
    user_id The Id of the User

    Required permission level

    Admin with can_unban Rights.

    Tokens

    Getting your own Token

    You can directly create your token with Userge AntiSpam Bot.

    HTTP Request

    POST https://api.userge.live/token

    Headers

    Parameter Description
    Content-Type application/json

    URL Parameters

    Parameter Description
    user_id Id of yours
    name name of give user id

    Getting about own token

    HTTP Request

    GET https://api.userge.live/token/self?api_key=$token

    Parameter Description
    api_key Your Api Key

    Promoting a Token

    HTTP Request

    POST https://api.userge.live/promotetoken

    Headers

    Parameter Description
    Content-Type application/json

    URL Parameters

    Parameter Description
    api_key Your Api Key
    user_id Id of yours
    perms perms, add space to add mutliple perms.

    Required Permission Level

    Admin with can_promote Rights.

    Demoting a Token

    This endpoint will demote a token to User Type.

    HTTP Request

    POST https://api.userge.live/demotetoken

    Headers

    Parameter Description
    Content-Type application/json

    URL Parameters

    Parameter Description
    api_key Your Api Key
    user_id Id of yours

    Required Permission Level

    Admin with can_demote Rights.

    Deleting a Token

    HTTP Request

    DELETE https://api.userge.live/token

    Headers

    Parameter Description
    Content-Type application/json

    URL Parameters

    Parameter Description
    api_key Your Api Key
    user_id Id of yours

    Misc

    Getting the Api version

    This returns the major, minor, micro and patch version of the API. This endpoint doesn't need a Authorization Token.

    HTTP Request

    GET https://api.userge.live/version

    Getting stats of API

    This returns the total number of bans in Userge Api. This endpoint doesn't need a Authorization Token.

    HTTP Request

    GET https://api.userge.live/stats

    Errors

    The Userge API uses the following error codes:

    Error Code Meaning
    400 Bad Request
    401 Unauthorized
    403 Forbidden
    404 Not Found
    405 Method Not Allowed
    500 Internal Server Error
